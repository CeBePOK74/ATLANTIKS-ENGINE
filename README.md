🚀 Atlantiks Engine (Evolution)
Универсальный поисковой движок на Go для мониторинга тендеров, заказов и маркетплейсов в режиме реального времени.

Проект Atlantiks-Evolution — это высокопроизводительный инструмент, созданный для профессионалов в сфере отделки и строительства. Он позволяет мгновенно находить новые заказы и тендеры, обходя защиты современных веб-ресурсов.

🛠 Технологический стек (Архитектура)
Процесс полностью контейнеризирован и оптимизирован под многопоточное выполнение на мощном оборудовании (рекомендовано: системы на базе Xeon E5 с 64GB+ RAM).

Core: Golang (эффективность и скорость горутин).

Engine: Playwright Go (автоматизация браузера Chromium внутри Docker).

Database: PostgreSQL (надежное хранение найденных данных).

Environment: Docker & Docker Compose (запуск в один клик без установки зависимостей).

AI Ready: Поддержка интеграции с локальными моделями Qwen и DeepSeek через OpenAI-совместимый API.

✨ Ключевые преимущества
⚡ Скорость Go: В десятки раз быстрее решений на Python за счет компиляции в бинарный код.

🕵️ Невидимость: Playwright эмулирует действия реального человека, позволяя парсить Авито, СБИС и Wildberries.

📦 Изоляция: Все системные библиотеки и браузеры уже находятся внутри Docker-образа.

💎 Чистые данные: Автоматическая фильтрация «шума» и классификация тендеров.

🏗 Масштабируемость: Легко добавлять новые площадки через расширение engine.go.

🚀 Быстрый старт
1. Требования
Установленный Docker и Docker Compose.

Git.

2. Установка
Bash
git clone https://github.com/CeBePOK74/ATLANTIKS-ENGINE.git
cd ATLANTIKS-ENGINE
3. Настройка
Отредактируйте параметры поиска в файле engine.go или через конфигурационные файлы в папке /config.

4. Запуск
Bash
docker-compose up --build
Система сама развернет базу данных, скачает Chromium и начнет мониторинг.

🧠 Интеллектуальный анализ (AI Powered)
Система подготовлена к глубокому анализу ТЗ тендеров с помощью локальных нейросетей:

Сбор текста: Робот забирает полное описание заказа.

Анализ: Передача данных в локальную LLM (например, Qwen 3.5).

Вердикт: Классификация релевантности заказа под конкретные навыки (плиточные работы, фасадные элементы и т.д.).
